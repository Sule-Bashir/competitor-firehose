## How to Run

### Option 1: Full Web Dashboard (Recommended)
```bash
# Clone the repository
git clone https://github.com/Sule-Bashir/competitor-firehose.git
cd competitor-firehose

# Run the web server
python3 working.py

# Open your browser to:
http://127.0.0.1:8080
```

### Option 2: Terminal Only
```bash
# Run the Python scraper
python3 final_submission.py
```

### Option 3: Simple HTTP Server
```bash
# Start a simple server
python3 -m http.server 8080

# Then open browser to:
http://127.0.0.1:8080/dashboard.html
```
